# Classification Models

A collection of Machine Learning classification models built with Python and scikit-learn.

## 📁 Models

| Model | Dataset | Description |
|-------|---------|-------------|
| 🌳 Decision Tree | Heart Disease | Predicts heart disease presence |
| 📍 K-Nearest Neighbors | Iris | Classifies iris flower species |
| 📈 Logistic Regression | Titanic | Predicts passenger survival |
| 🌲 Random Forest | Diabetes | Predicts diabetes likelihood |

## 🛠️ Tools Used

`Python` `scikit-learn` `Pandas` `Matplotlib` `Jupyter Notebook`
